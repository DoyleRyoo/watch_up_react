# watch_up_react
Stock Market Simulator React UI

## 의존성
```
React + TypeScript + Vite
├── React Router
├── Axios
├── TanStack Query
├── Zustand
├── React Hook Form
├── Zod
├── Tailwind CSS 3
├── Recharts
├── Lightweight Charts
├── Lucide Icons
├── Day.js
├── Toast
├── ESLint
├── Prettier
└── Vitest + Testing Library
```
``` bash
npm install react-router axios @tanstack/react-query @tanstack/react-query-devtools zustand react-hook-form zod @hookform/resolvers recharts lightweight-charts lucide-react dayjs clsx tailwind-merge class-variance-authority react-hot-toast
```
``` bash
npm install -D tailwindcss@3 postcss autoprefixer prettier prettier-plugin-tailwindcss eslint-config-prettier vitest jsdom @testing-library/react @testing-library/jest-dom @testing-library/user-event @types/node
```
